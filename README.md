## Маппинги

### В config/lsp.lua
* gd            - переход к определению
* gD            - переход к объявлению
* gi            - переход к реализации интерфейса
* gy            - переход к определению типа
* gr            - показать все использования
* <C-k>         - показать сигнатуру функции
                  (аргументы) во всплывающем окне
* K             - информация о слове под курсором
* \<leader\>rn  - переименовать символ
* \<leader\>ws  - поиск символа по рабочему 
                  пространству
* \<leader\>ds  - поиск символа в текущем документе
* \<leader\>f   - форматирование 
* \<leader\>ca  - открыть меню code actions 
                  (быстрые исправления)
* [d            - предыдущая ошибка
* ]d            - следующая ошибка
* \<leader\>e   - окно с деталями диагностики
* \<leader\>q   - показать диагностику 
                  для текущей строки
* \<leader\>dd  - открыть quickfix 
                  со всеми ошибками проекта

### В config/mappings.lua

\<leader\> = ' '

* \<leader\>l   - вправо
